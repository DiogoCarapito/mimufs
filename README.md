# mimufs

Mim@uf Business Intelligence Python Library for Portuguese Primary Care

## Cheatsheet

### Build package

```bash
python3 setup.py sdist bdist_wheel
```

### Use twine to upload to PyPI
```bash
twine check dist/*
twine upload dist/*
```
